Установка утилиты сборки пакетов linex
 	$  sudo apt install checkinstall build-essential autoconf automake
 Войти в папку с архивом
 	$ cd ~/путь/к/папке/с/архивом 	
 Извлечь архив
  tar -xvf имя_архива.tar.gz
  Перейти в папку с только что полученными исходными файлами:
  	$ cd ./имя_архива
  Создание скрипта configure
