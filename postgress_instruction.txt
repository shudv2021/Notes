1) Узнать текущую версию PostgreSql
способ 1 - $ postgres -v (сработает только при соответствующей настрйлу PATH)

Способ 2 – Узнать где находится PSQL 
	$ sudo find /usr -wholename '*/bin/postgres'
	или
	sudo updatedb
	locate bin/postgres
	Полуаем путь перейдя по которому можно запскать команды 
2)Узнать текущего пользователя $ whoami
имя пользователя совпадает с именем текущей базы данных
если нет такой базы данных использовать стартовую базу $psql postgres
Выход из консоли \q
